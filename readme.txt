== Description ==

YoApy Social Poster is a complete WordPress plugin that lets you schedule and automatically publish your posts to multiple social networks. With a modern and intuitive interface, it offers direct integration with Facebook, Instagram, YouTube, and TikTok through the YoApy API.

**Key Features:**

* **Visual Planner**: Modern interface with real-time post preview
* **Multiple Social Networks**: Support for Facebook, Instagram, YouTube, and TikTok
* **Flexible Scheduling**: Publish immediately or schedule for specific times
* **Integrated Metabox**: Publish directly from the WordPress post editor
* **Content Types**: Support for images, videos, stories, and live streams
* **Detailed Logs**: Track the status of all publications
* **Smart Preview**: See how your post will look on each social network
* **Responsive Interface**: Optimized design for desktop and mobile

**How It Works:**

1. **Simple Setup**: Add your free YoApy API keys (available at yoapy.com)
2. **Accounts per Network**: Configure different accounts for each social network
3. **Publish**: Use the standalone planner or the WordPress metabox
4. **Track**: Monitor the status of all posts in detailed logs

**Technical Features:**

* Interface built with Tailwind CSS
* AJAX for real-time updates
* Cron job system for scheduling
* Integration with WordPress Media Library
* Complete data sanitization and validation
* Compatible with WordPress 5.0+

== Installation ==

**Automatic Installation:**
1. In WordPress admin, go to Plugins > Add New
2. Search for "YoApy Social Poster"
3. Click "Install Now" and then "Activate"

**Manual Installation:**
1. Download the plugin
2. Extract the ZIP file
3. Upload the folder to `/wp-content/plugins/`
4. Activate the plugin from the WordPress Plugins menu

**Configuration:**
1. Go to YoApy Planner > Settings
2. Create your free keys at [yoapy.com](https://yoapy.com)
3. Paste your Key ID and Secret
4. Configure your social network accounts
5. Test the connection
6. Start publishing!

== Frequently Asked Questions ==

= Do I need to pay to use YoApy? =

No! YoApy provides free keys you can get at yoapy.com. There are also paid plans with additional features.

= Which social networks are supported? =

Currently: Facebook, Instagram, YouTube, and TikTok. More networks are coming soon.

= Can I schedule posts for different times? =

Yes! You can publish immediately or schedule for any future date/time.

= Does the plugin work with any WordPress theme? =

Yes, it is compatible with all themes. The admin interface is independent from the theme.

= How do I auto-publish my WordPress posts? =

Use the "YoApy Social Poster" metabox in the post editor. Configure the desired networks and check "Send when publishing".

= Can I use different accounts for each social network? =

Yes! You can set a default account and specific accounts per network.

= What happens if a publication fails? =

= Does the plugin affect site performance? =

No! Posts are processed in the background via cron jobs, without slowing down the site.
